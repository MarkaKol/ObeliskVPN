Obelisk VPN - Chrome Extension

Overview

Obelisk VPN is a Chrome extension that provides secure and private internet access through a VPN proxy service. It features a modern, visually appealing interface with gradient backgrounds and smooth animations.

Features
Secure VPN Connection: Connect to VPN servers using SOCKS5 protocol

Automatic Server Selection: The system automatically assigns the optimal server for your location

Modern UI: Beautiful gradient design with animated elements

Multi-language Support: 14 languages available including English, Russian, Chinese, Spanish, French, and more

Connection Status Tracking: Real-time monitoring of your VPN connection

User-Friendly Interface: Simple one-click connect/disconnect functionality

Installation
From Chrome Web Store
Visit Chrome Web Store

Search for "Obelisk VPN"

Click "Add to Chrome"

Confirm the installation

Manual Installation (Developer Mode)
Download or clone this repository

Open Chrome and navigate to chrome://extensions/

Enable "Developer mode" (top right toggle)

Click "Load unpacked"

Select the extension folder

Usage
Click the Obelisk VPN icon in your Chrome toolbar

The popup will load with animated preloader

Click the "Connect" button to establish VPN connection

Your connection status will be displayed in real-time

Click "Disconnect" to end the VPN session

Technical Details
Proxy Protocol: SOCKS5

Connection Type: XRay Reality protocol

Server Assignment: Automatic based on user location and server health

Storage: Local storage for user preferences and connection status

API Endpoint: Backend server for server management and assignment

Permissions
The extension requires the following permissions:

proxy: To configure VPN proxy settings

storage: To save user preferences and connection status

scripting: For extension functionality
